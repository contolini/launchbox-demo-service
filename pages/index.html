Hello! Welcome to Launchbox

This site is powered by the Launchbox Demo Service. If you've just installed Launchbox for the first time,
seeing this means that everything is working as expected!

Routes

The launch.yaml file for this service defines three routes:

route: pages

This static route serves files from a directory. 

pages:
    route: /
    type: static
    options:
        path: pages

Used by: this page

route: api

This WSGI route serves a python3 bottle app. 

api:
    route: /api
    type: wsgi
    options:
        path: app/api.py
        var: app

Used by: the sample API

route: assets

This static route serves a specific JSON file. 

assets:
    route: /static/data.json
    type: static
    options:
        path: static/data.json

Used by: the static JSON endpoint
